!!Renaming Tool

This page demonstrates how the renaming tool functions. Just enter the search word in the input field below,enter an replacement and press the preview button. Befor any replacement is made, you see a list of findings from which you can choose which findings should be replaced with the entered replacement.

[{KnowWEPlugin renamingtool}]

The renaming tool also provides some advanced replacement settings. For this click on the "__Further settings__" link. There you can set the following things.
* only consider uppercase or lowercase words
* the search word has a certain context (prefix or suffix) in which it occurs
* you can specify the type of the section in which the word accurs .e.g use this for only replace something for example in __RuleSections__ etc.
